# tulpa-site

Marketing site for [Tulpa](https://tulpa.run) — *it runs the account, you don't.*

Static landing page served by a zero-dependency Node server.

```bash
npm start   # serves ./site on $PORT (default 3000)
```

Deploy: any host that runs `npm start` (Railway, etc.). Custom domain → tulpa.run.
